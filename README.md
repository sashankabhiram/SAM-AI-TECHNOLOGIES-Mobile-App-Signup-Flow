# 📱 Mobile App Signup Flow UI Design

## 📌 Overview

This project was completed as **Task 4** during my **UI/UX Design Internship at SAM AI Technologies**.

The objective was to design a seamless and user-friendly mobile app signup flow that provides an intuitive onboarding experience. The design focuses on simplicity, clear navigation, and an engaging interface while following modern UI/UX principles.

---

## 🎯 Objective

* Design a complete mobile app signup flow.
* Create an intuitive onboarding experience.
* Ensure smooth navigation between connected screens.
* Apply user-centered design principles.
* Maintain a clean and modern visual interface.

---

## ✨ Features

* Welcome Screen
* Sign Up Screen
* Login Screen
* User Information Form
* Input Validation Layout
* Call-to-Action Buttons
* Connected User Flow
* Mobile-First Design
* Clean and Minimal UI
